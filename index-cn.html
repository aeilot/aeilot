#项目


Daily Notes 是我第一个 Android 项目，是个功能强大的记事本应用。可以在 酷安 下载到。除了基本的记事本功能外，它还包含即时渲染、多标签页、人性化搜索功能、标签分类、WebDAV、应用锁等功能。


ProblemSet 里面是我 OI 刷题的题库。我不仅仅弄 OI，目前是文化课选手，所以更新不是很频繁。


GitHub Dark Mode Safari 是我帮助 Mr Will 移植的浏览器插件，现在由于官方出了相同的功能，已经停止更新了。


Hexo Theme Paperwhite 是我为 EN Blog 定制的 Hexo 主题，极简主义。


lsnotes 可以显示当前目录下的 .lsnotes 文件，实际上 这有无限可能性等你来探索！最开始的想法只是 ls 的时候显示文件夹信息。


How To Use Search Engines 是我创作的 语言简洁的搜索引擎教程。（水项目 Doge
