HSEFZ Digital Life is the online system made with Django designed for my school! It's actively maintained and currently passed on to the next generation of technicians! The system incorporates features including club selection, volunteering stats and sports league scoring, etc.


HSEFZ Club Yearbook is an online club catalog, featuring live edit with Notion. Designed for club showcase for my school.


ProblemSet is where I store all the solutions to the OI problems I've solved.


GitHub Dark Mode Safari is made for Mr Will, which is a port of his project, migrating the Chrome theme to Safari with a nice native macOS UI. As GitHub rolled out its official dark mode features, the project is now archived.


Ancient Chinese Dict CLI is a CLI dictionary app for ancient Chinese studies.


Hexo Theme Paperwhite is customized for my blog, which features minimalist design.


Handy Scripts is the collection of scripts I created to help with school technical work.


lsnotes displays the .lsnotes file in the current directory. Actually it has endless possibilities remaining to be unraveled!


How To Use Search Engines is a tutorial to search engines targeted at teenagers.


timqian/sideproject.guide is a tutorial to side projects. I helped with the English translation.
